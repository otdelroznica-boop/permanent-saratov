Контакты
Контакты используются везде на сайте и в SEO-разметке.
Экспорт сайта
Скачайте готовый файл и загрузите на хостинг.
1
Настройте шаблон, фото и тексты в панели, затем нажмите «Сохранить»
2
Нажмите «Скачать сайт» — получите файл index.html со всеми настройками внутри
3
Загрузите файл на GitHub в репозиторий. Папку img/ тоже загрузите отдельно
4
Сайт обновится на permanent-saratov.ru в течение 1–2 минут
⚠️ Фото из папки img/ нужно загрузить отдельно на GitHub. Или используйте вкладку «Фото» — тогда они встроятся прямо в HTML файл.
Настройки
Одноразовая настройка для публикации сайта напрямую.
🔑 Токен GitHub нужен один раз — он сохраняется в браузере и позволяет кнопке «Опубликовать» обновлять сайт автоматически без загрузки файлов.
💬 Комментарии ВКонтакте
Создайте приложение на vk.com/editapp → «Встраиваемое приложение» → укажите домен permanent-saratov.ru. Скопируйте ID приложения и вставьте ниже.
После сохранения кнопка «🚀 Опубликовать» будет обновлять сайт за 1–2 минуты.
